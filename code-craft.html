৬ মাস মেয়াদি প্রোগ্রামিং কোর্সে সম্পূর্ণ কোর্সটি মোট ৪ টি মডিউল এবং ৪৮ টি ক্লাসে ভাগ করা হয়েছে , যেখানে আপনার সন্তান ৫/৬ টি পূর্ণাঙ্গ প্রজেক্ট বাস্তবে লাইভ করতে পারবে ।

মডিউলে ১ এ রয়েছে Introduction to HTML এর উপর ১২ টি ক্লাস যেখানে Basics of HTML : Structure and Syntax | Elements and Tags | Images and Multimedia | Designing Tables for Data Presentation সম্পর্কে পূর্ণাঙ্গ ধারণা পাবে।

মডিউলে ২ এ রয়েছে Introduction to CSS এর উপর ১২ টি ক্লাস যেখানে Inline vs. Internal vs. External Stylesheets | Styling Boxes and Layouts | Adding Gradients and Images as Backgrounds সম্পর্কে জানতে পারবে।

মডিউলে ৩ এ রয়েছে Introduction to JavaScript এর উপর ১২ টি ক্লাস যেখানে Conditional Loops and Loop Control | Functions in JavaScript | Modifying HTML and CSS Dynamically | Exploring jQuery for DOM Manipulation সম্পর্কে ধারণা পাবে।

মডিউলে ৪ এ রয়েছে Python Fundamentals এর উপর ১২ টি ক্লাস যেখানে Conditional Statements in Python | Using Loops for Iteration in Python | Lists, Dictionaries, and Tuples | Defining Classes and Objects in Python সম্পর্কে বিস্তার ধারণা পাবে।
